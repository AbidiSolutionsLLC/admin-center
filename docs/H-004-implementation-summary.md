## API Endpoints Used

| Method | Endpoint | Purpose |
|--------|----------|---------|
| GET | `/api/v1/people` | Fetch all users (with optional filters) |
| GET | `/api/v1/organization` | Fetch departments for filter dropdown |

| POST | `/api/v1/people/bulk-invite` | Bulk invite from CSV |
| PUT | `/api/v1/people/:id` | Update user profile |
