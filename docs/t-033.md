        - Tabbed modal interface (Credentials & Sync | Field Mapping)

     3. "Sync Now" triggers sync endpoint and shows result toast
        - POST /integrations/:id/sync endpoint

        - useSyncNow hook with toast on success/failure
        - Toast shows statistics: "Sync completed: X records processed, Y created, Z updated"
        - Integration status updated to reflect sync result

     4. Sync log appears after every sync (manual or scheduled)
        - IntegrationSyncLog model with full tracking (started_at, completed_at, duration_ms,
          records_processed/created/updated/failed)
        - GET /integrations/:id/sync-logs endpoint with pagination
        - SyncLogViewer component with table view of all sync operations
        - Error messages displayed for failed syncs
        - Status badges (Success/Failed/Partial) with color coding

     5. Sync failure triggers admin notification (use Notification system from H-031)
        - notifySyncFailure() function in controller
        - Currently logs audit event integration.sync_failed_notification (ready for H-031 integration)
        - TODO comment marks where Notification service will be integrated
        - Sync failure visible in UI via error toast and sync log viewer

     6. Disconnect wipes `credentials_encrypted` field
        - POST /integrations/:id/disconnect endpoint
        - Sets credentials_enc = '' (wipes encrypted data)
        - Updates status to disconnected, sets disconnected_at
        - Audit event integration.disconnected logged with before/after state
        - UI confirmation dialog: "This will wipe all stored credentials"

     7. All mutations produce audit events
        - integration.connected - on creation
        - integration.updated - on settings update
        - integration.synced - on sync completion (with stats)
        - integration.disconnected - on disconnect (with credential wipe confirmation)
        - integration.field_mapping_updated - on field mapping save
        - integration.sync_failed_notification - on sync failure (notification trigger)
        - All audit events include actor info, IP, user agent, before/after state
