    ✓ RBAC Engine Resolves Conflicts: Deny Overrides Grant

    Core Logic (`lib/rbac.ts`):
     - `resolveUserPermissions()` - Resolves all permissions for a user across all their roles
     - Two-Pass Algorithm:
       1. First pass: Collect all grants from all roles
       2. Second pass: Apply denies (deny always overrides grant)
     - Result: Map<string, boolean> with effective permissions

    Test Verification:
     - Test user assigned both "HR Admin" (grants people:create:all) and "Employee" (denies people:create:all)
     - Result: Permission is DENIED ✅ - Deny overrides grant working correctly
